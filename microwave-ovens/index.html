Microwave ovens, an unnecessarily bad design

Does it take more time to set up the machine than to warm your food?

The only thing that I need to set is the time that my food should cook. I do not need more controls.

Any extra button will make it harder to figure out, which controls to use to set the time. Maybe I can get used to and remember the right buttons, but why not make it stupidly simple?

How to fix it

Have a turning knob like a cooking timer.

Maybe add a second turning knob for setting the power, but it should by styled in a way that the timer knob is the only important control.
